ArcGIS web map schema

Definition

The ArcGIS web map is a common specification for ArcGIS web and mobile applications. The web map exposes functionality contained in the web/mobile APIs. Although it can be read by the web/mobile APIs, it is not created directly with the web API. It can also be read by ArcGIS Desktop and ArcGIS Explorer Desktop.

Folders

  • src folder contains all the web map schema.

  • doc folder contains markdown files for all the web map schema.

  • schema2cpp folder contains scripts that generate class definition and implementation in C++ from json schema.

  • tool folder contains a python script that converting a JSON schema file to a markdown file.
